pub enum Cocons<H, T> {
    Head(H),
    Tail(T),
}
Expand description

Cons is a node in a list. Cocons is the dual of the cons. Cons is both head and tail, cocons is either a head or a tail.

Variants

Head(H)

Head of the coproduct (i.e. first element in the type-list).

Tail(T)

Tail of the coproduct (i.e. not the first element in the type-list).
